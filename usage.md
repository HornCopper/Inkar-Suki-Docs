# Inkar Suki 使用文档

欢迎来到 `Inkar Suki` 使用文档。

本文档用于整理音卡当前可用的主要功能和命令。命令参数约定如下：

- 尖括号 `<...>` 表示必填参数。
- 方括号 `[...]` 表示可选参数。
- 使用命令时不要把尖括号或方括号本身一起发送。
- 若参数为 `<服务器>`，且本群已绑定服务器，通常可以省略服务器。
- 部分功能依赖外部接口、群权限或个人权限；若命令提示不可用，请以实际提示为准。
